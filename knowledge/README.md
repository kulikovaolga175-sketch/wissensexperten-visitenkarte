# knowledge/ — справочники и руководства

Здесь хранятся файлы-инструкции, которые Агент создаёт когда одна тема всплывает 2+ раза.

## Когда создавать файл в knowledge/

- Тема, процесс или инструкция нужна регулярно
- Документ слишком длинный для MEMORY.md
- Нужен чек-лист или пошаговое руководство

## Формат

`knowledge/тема.md` — короткое английское имя файла, понятное название.

После создания — добавь ссылку в раздел «Ссылки на knowledge/» в `MEMORY.md`.
