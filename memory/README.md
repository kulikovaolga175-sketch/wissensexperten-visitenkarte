# memory/ — дневник работы по дням

Агент создаёт файлы в формате `YYYY-MM-DD.md` для отслеживания прогресса каждой сессии.

## Шаблон дневного файла

```
# YYYY-MM-DD

## Сделано
- 

## Решения
- 

## TODO на потом
- 
```

## Правила

- Дневник — рабочий журнал, не отчёт. Пишите коротко, телеграфно.
- Долгосрочные факты об Ольге — в `MEMORY.md`, не сюда.
- Справочники и инструкции — в `knowledge/`, не сюда.
- История за день = этот файл.
